## Some CPED Reset Flags

To set Reset Ped Flags call it every frame.

## Example

```lua
Citizen.CreateThread(function()
  while true do
    Citizen.Wait(0)
    Citizen.InvokeNative(0xC1E8A365BF3B29F2,PlayerPedId(),2,true)   -- SetPedResetFlag, flag 2 is preventing ped to jump
  end
end)

  while true do

  end
end)
